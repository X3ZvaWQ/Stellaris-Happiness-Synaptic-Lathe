## 简介

本模组移除了玩家将 **主体物种（Owner Species）** 送入 **突触凝练机（Synaptic Lathe）** 享乐时，来自其他帝国的 **“种族灭绝者”** 或者 **“净化我方物种”** 外交评价。

## 功能

- **平衡性**：仅将主体物种送入突触凝练机不再触发种族灭绝的外交评价（包括放出来的附庸），但是烧别个种族依旧会触发种族灭绝。
- **兼容性**：适用于群星 4.2+ 版本（需《机械纪元》DLC）。
- **存档兼容**：可随时加入现有存档。
